2 Drift Theorems

2.1 Additive drift

[Additive drift theorem] If \(\mathbb {E}[X_{t+1} - X_t \mid X_t] \geq \delta {\gt} 0\) for \(X_t \notin T\), then \(\mathbb {E}[\tau ] \leq X_0 / \delta \). DriftTheorems.AdditiveDrift.additive_drift_theorem def:expected_value

2.2 Multiplicative drift

[Multiplicative drift theorem] If \(\mathbb {E}[X_{t+1} \mid X_t] \leq (1 - \delta ) X_t\) for \(X_t \notin T\), then \(\mathbb {E}[\tau ] \leq \frac{\ln (X_0)}{\delta }\). DriftTheorems.MultiplicativeDrift.multiplicative_drift_theorem def:expected_value

2.3 Negative drift

[Negative drift theorem] If \(\mathbb {E}[X_{t+1} - X_t \mid X_t] \leq -\delta {\lt} 0\) and bounded variance, then \(\tau \) is exponentially large. DriftTheorems.NegativeDrift.negative_drift_theorem def:expected_value, lem:hoeffding
